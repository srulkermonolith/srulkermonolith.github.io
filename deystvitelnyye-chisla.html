Квадратный корень из числа

Квадратным корнем из числа а называют число, квадрат которого равен а.
Например, квадратными корнями из числа 16 являются числа 4 и -4, по- скольку 42 = 16 и (-4)2 = 16.
Из отрицательного числа квадратного корня не существует.
Арифметическим квадратным корнем из числа а называют неотрицательное число, квадрат которого равен а.
Например, арифметическим квадратным корнем из числа 36 является число 6. Для обозначения квадратного корня используют знак √.
Запись m читают так: «Арифметический квадратный корень из m» или просто: «Квадратный корень из m», при этом число m называют подкоренным выражением.
Например: 144 = 12; 0 = 0
Действие, посредством которого находится квадратный корень, называется извлечением квадратного корня.
Итак, определение квадратного корня можно записать следующим образом: если а ≥ 0 и b2 = а, то a = b
Из этого определения следует, что a2 = a, если a ≥ 0
Например: 1,72 = 1,7, но выражение -32 не имеет смысла.
